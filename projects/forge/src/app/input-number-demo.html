Numeric input

A dedicated numeric field whose model is a real number, formatted per locale via Intl.NumberFormat. The first two fields bind the same value — type in either and watch it reformat in the other's locale; grouping lands on blur, never mid-keystroke. The model never becomes a "1,234" string.
