Order list

Reorder the dashboard sections by dragging a row, or select one or more (click, Ctrl+click, Shift+click; or Space, Shift+Arrow, Ctrl+A by keyboard) and use the move buttons to move the whole set — every move is announced to a screen reader. Arrow keys move focus; the list is a multi-selectable role="listbox", so it is fully operable without a pointer. Its visible header also names the list for a screen reader. Billing is locked — dimmed and non-movable, but still keyboard-navigable. Type in the filter box to narrow the sections by name — reorder is disabled while filtering, since a partial view has no coherent "move up". Toggle controls after list to move the button column to the other side (it mirrors for free under RTL), or drag handle to drag rows only by their grip.
