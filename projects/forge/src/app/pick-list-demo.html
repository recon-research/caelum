Pick list

Assign roles by dragging one across, or multi-select roles — click, Ctrl+click, Shift+click, or Space / Shift+Arrow / Ctrl+A by keyboard — and move the whole block with the transfer buttons between the lists. Reorder a list in place by dragging a row or using its outer up / top / down / bottom buttons; every move is announced to a screen reader. Type in a list's filter box to narrow it by name — reorder is disabled while a list is filtered, but transfer stays live. Each list carries a projected header that is also its accessible name, and is a role="listbox", so the whole picker is operable without a pointer. Toggle right-to-left to see the transfer glyphs mirror (the panes flip for free).
