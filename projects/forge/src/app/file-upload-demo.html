File upload

Choose files with the keyboard (Tab to the control, then Enter) or drop them on the zone. Images up to 512 KB are accepted; anything larger or non-image is rejected before upload at the trust boundary and announced to a screen reader. Each accepted image shows an inline thumbnail preview. Wiring [url] turns on the HttpClient progress / cancel / retry upload.
